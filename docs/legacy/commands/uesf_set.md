# uesf set

```bash
uesf set --help
```

```
Usage:  uesf set [OPTIONS]

Set a global configuration value for UESF. This value will be stored globally to modify default UESF behaviors.

Options:
  --help  Show this message and exit.
  --data-dir=<path>  Set the root directory to store UESF internal database and preprocessed datasets. (Default is ~/.uesf/data)
```
